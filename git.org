* Setting up Git
** SSH Key Pairs
Generate a key-pair using a modern, and secure, algorhythm
#+begin_src shell
  ssh-keygen -t ed25519 -C "cygnetix@example.com"
#+end_src

When loading the key, to push changes to git, use ~ssh-add~.
#+begin_src bash
  ssh-add ~/.ssh/id_ed25519
#+end_src
