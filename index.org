* Welcome
This, my friends, is a very minimalistic [[https://maggieappleton.com/garden-history][digital garden]], created using [[https://www.gnu.org/software/emacs/][Emacs]]. Put another way; it's a living breathing mess of my personal notes and configuration files published to the internet for the world to see.

* Config Files
Emacs is a highly customisable editor, so there's lot of setup that goes into defining the perfect workflow. What follows is 100% of my own Emacs configuration. I tend to try and use the build-in packages as much as possible, but some external dependencies are unavoidable.

- [[file:emacs_base_packages.org][Emacs Base]] :: A very basic emacs configuration that I use with miniaml external dependencies. The goal is to have an environment that's fast and uses many of the excllent default packages that now ship with modern releases of emacs.
- [[file:emacs_external_packages.org][Emacs External]] :: These are external packages installed through MELPA. I like to distinish these from config used for packages that are provided out of the box.
- [[file:emacs_golang.org][Go]] :: Golang is my prefered language for when I want to write something that's both robust and easily shared with others. It strikes a great balance between performance and simplicity while having many modern features built-in.
