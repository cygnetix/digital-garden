Config Files

Emacs is a highly customisable editor, so there's lot of setup that goes into defining the perfect workflow. What follows is 100% of my own Emacs configuration. I tend to try and use the build-in packages as much as possible, but some external dependencies are unavoidable.

Emacs Base
A very basic emacs configuration that I use with miniaml external dependencies. The goal is to have an environment that's fast and uses many of the excllent default packages that now ship with modern releases of emacs.
Emacs External
These are external packages installed through MELPA. I like to distinish these from config used for packages that are provided out of the box.
Claude Code
Making Claude and Emacs play nice together. I realise the value in having Claude at your finger tips, integrated seamlessly straight into Emacs.
Go
Golang is my prefered language for when I want to write something that's both robust and easily shared with others. It strikes a great balance between performance and simplicity while having many modern features built-in.
